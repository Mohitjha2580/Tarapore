# 🎓 Tarapore School - Student Engagement Platform

**"Where Learning Comes Alive! Explore it! Experience it! And Excel in it while igniting curiosity!"**

A comprehensive student engagement platform built for Tarapore School, Jamshedpur, designed to inspire students and track their holistic development through the S.M.A.R.T framework.

## 🏫 About Tarapore School

**S.M.A.R.T Philosophy:**
- **S**piritually inclined
- **M**orally upright  
- **A**rdent achievers
- **R**esponsible citizens
- **T**houghtful and tolerant

## ✨ Platform Features

### 🎯 Core Features
- **Live Leaderboards** - Real-time class and house rankings
- **Achievement Badge System** - Visual rewards for various accomplishments
- **Activity Tracker** - Monitor participation in debates, arts, sports, volunteering, and innovation
- **Wall of Fame** - Spotlight top performers and achievements
- **Teacher Dashboard** - Simple point assignment system
- **Student Profiles** - Personal progress tracking and goals
- **Daily Engagement** - Login streaks and daily challenges

### 🎨 Design Excellence
- **Mobile-First Design** - Perfect experience on phones and tablets
- **Modern UI/UX** - Clean, intuitive interface with smooth animations
- **Accessibility** - Keyboard navigation and focus indicators
- **Interactive Elements** - Hover effects, progress animations, modal popups

### 🎨 Color Palette
- **Primary:** Deep blues (#2c3e50, #3498db) - Trust and wisdom

## 🚀 Quick Start

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No server setup required - runs entirely client-side

### Installation
1. Clone or download the repository
2. Open `index.html` in your web browser
3. Start exploring the platform!

```bash
# Clone the repository
git clone <repository-url>
cd tarapore-school-platform

# Open in browser
open index.html
```

## 🎨 Customization

### Color Variables
All colors are defined in CSS custom properties for easy customization:

```css
:root {
    --primary-dark: #2c3e50;
    --primary-blue: #3498db;
    --secondary-green: #27ae60;
    --secondary-light-green: #2ecc71;
    --accent-purple: #9b59b6;
    /* ... additional variables */
}
```

### Adding New Features
1. **HTML Structure** - Add new sections to `index.html`
2. **CSS Styling** - Follow existing design patterns in `styles.css`  
3. **JavaScript Logic** - Extend functionality in `script.js`
